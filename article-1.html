Assembly /bin/sh shellcode

We are going to create an assembler code that will allow us to execute a linux shell: /bin/sh.

Let's start :

Assembly code :

section .text global _start _start: push 59 <-- push 59 on the stack (59 = execve syscall) pop rax <-- pop rax with 59 xor rsi, rsi <-- set rsi to 0 push rsi <-- push rsi on the stack mov rdx, "//bin/sh" <-- put //bin/sh into rdx register push rdx <-- push rdx register on the stack xor rdx, rdx <-- set rsi to 0 mov rdi, rsp <-- set pointer syscall <-- call kernel

Let's compile code :

nasm -f elf64 shell.asm -o shell.o

ld shell.o -o shell

./shell

$

Fine it work !
